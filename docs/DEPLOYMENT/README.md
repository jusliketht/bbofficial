# Deployment Documentation

This directory contains deployment guides and configuration files for the Burnblack ITR platform.

## Contents

- **[VERCEL_DEPLOYMENT_GUIDE.md](./VERCEL_DEPLOYMENT_GUIDE.md)** - Comprehensive guide for deploying to Vercel
- **[ENVIRONMENT_VARIABLES_TEMPLATE.md](./ENVIRONMENT_VARIABLES_TEMPLATE.md)** - Template for all required environment variables

## Quick Start

1. Read the [Vercel Deployment Guide](./VERCEL_DEPLOYMENT_GUIDE.md)
2. Set up your environment variables using the [Environment Variables Template](./ENVIRONMENT_VARIABLES_TEMPLATE.md)
3. Follow the step-by-step deployment instructions

## Deployment Platforms

- **Vercel** (Recommended) - [Deployment Guide](./VERCEL_DEPLOYMENT_GUIDE.md)
- More platforms coming soon...

## Prerequisites

Before deploying, ensure you have:

- ✅ Git repository set up
- ✅ Database (Supabase) project created
- ✅ All API keys and credentials ready
- ✅ Domain name (optional)

## Getting Help

If you encounter issues during deployment:

1. Check the [Troubleshooting](./VERCEL_DEPLOYMENT_GUIDE.md#troubleshooting) section
2. Review Vercel documentation
3. Check application logs in Vercel dashboard
4. Contact the development team
